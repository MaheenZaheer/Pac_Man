# Pac-Man Game (Java)

## Description

This is a simple **Pac-Man game implemented in Java**. The project is created for **learning and educational purposes** and demonstrates basic concepts of Java programming and simple game development using Java graphics.

This implementation is **based on the Pac-Man Java project by ImKennyYip**, and it may include small changes made for practice and understanding.

## Features

* Classic Pac-Man style gameplay
* Keyboard controls for player movement
* Maze-based game layout
* Basic ghost movement
* Pellet collection

## Technologies Used

* Java
* Java AWT / Swing (for graphics and UI)

## Requirements

To run this project, you need:

* Java JDK 8 or higher
* Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans) or Command Prompt

## How to Run the Project

### Method 1: Using an IDE

1. Download or clone the repository
2. Open the project in your Java IDE
3. Locate the main Java file (the file that contains `main` method)
4. Run the file

### Method 2: Using Command Line

1. Open Command Prompt / Terminal
2. Navigate to the project folder
3. Compile the Java file:

   ```bash
   javac pac_man.java
   ```
4. Run the game:

   ```bash
   java pac_man
   ```

## Purpose

This project was created to practice Java programming concepts such as:

* Object-Oriented Programming (OOP)
* Event handling
* Basic game logic
* Graphics rendering
